A Fourier Machine

This is very vague so far, but the rotating circles formulation makes me certain that there is a way to construct a machine to draw these shapes. It would likely use cogs. It would be something like a spirograph on steroids (aside: the spirograph is an amazing toy).

If it were to ever be remotely useful, except as a cool novelty, then the radii of the circles/ cogs would need to be adjustable.

Clearly I haven't worked it out yet, but it's a very exciting thought experiment.
